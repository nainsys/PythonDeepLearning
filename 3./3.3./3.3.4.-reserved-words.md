# 4.3.4.     예약어\(Reserved Words\)



다음 목록은 Python 키워드를 보여줍니다. 이것은 예약어이며 상수 또는 변수 또는 다른 식별자 이름으로 사용할 수 없습니다. 모든 파이썬 키워드는 소문자만 포함됩니다.

| and | exec | not |
| :--- | :--- | :--- |
| assert | finally | or |
| break | for | pass |
| class | from | print |
| continue | global | raise |
| def | if | return |
| del | import | try |
| elif | in | while |
| else | is | with |
| except | lambda | yield |
